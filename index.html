使い方

初回の手順

  1. SETTING で WordPress サイトと媒体テンプレートを登録します。
  2. WRITE でタイトルと本文を入力します。
  3. カテゴリーとタグは 更新 ボタンで同期できます。
  4. WordPress送信 ＆ 媒体成形 で WP送信と媒体テキスト生成を行います。
  5. HISTORY では送信済み記事の再編集と再利用ができます。

バックアップ／インポート

  1. バックアップ保存 で JSON を保存します。
  2. 別端末で使う場合は バックアップ読込 で復元します。
  3. 自動バックアップは最新分を 最新自動バックアップを保存 で出力できます。

WP下書きの一括取り込み

  1. SETTING の「WP下書きを履歴へ一括取り込み」から実行します。
  2. 対象を「下書き／公開済み／両方」から選べます。
  3. 既に履歴にある記事は自動でスキップされます。

アプリとして使う方法（簡易）

  1. ブラウザで開いたら、右上メニューから「ホーム画面に追加」します。
  2. アイコンから起動すればアプリ風に使えます。

FAQ

  • 投稿が反映されない → WordPressのアプリパスワードを再発行して再登録してください。
  • 履歴が空 → WP下書き取り込み またはバックアップ復元を実行してください。
  • スマホでも使える？ → 使えます。ホーム画面追加でアプリ風に動きます。
  • 任意プラットフォームのチェックは？ → SETTING の「任意プラットフォーム管理」で追加・削除できます。
